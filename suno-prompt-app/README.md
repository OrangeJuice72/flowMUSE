# SUNO Prompt Builder

A React + Vite app for building:
1. a lyrics guidance prompt for SUNO
2. a style / sound prompt for SUNO

## Setup

```bash
npm install
npm run dev
```

## Structure

- `src/data/*.js` contains the editable pools and genre profile logic
- `src/components/` contains UI building blocks
- `src/utils/helpers.js` contains suggestion and copy logic
